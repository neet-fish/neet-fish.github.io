我是心宿二，这是测试文章，也是第一篇博文

以下是2025/12/4日写的内容。

昨天晚上才弄好网站，没时间写点东西，想了一下就顺便接着第一篇的测试写下来吧。今天周四下午没课正好整理一下这两天折腾这玩意的心得。

现在已经安装好typora，写博文比记事本方便多了。

想弄个个人博客的想法有好久了，但真正打算实践也就是在前两天。我对这方面的知识为零，看了好多教程最后决定用hexo+github来搭建博客。

一开始还挺顺利，直到在本地部署时出错了，运行完npx hexo init 后按照教程应该是hexo install 但出错了，按照弹幕的不要hexo install ，而是cd blog ，但也不行。（悲

后来不知道干嘛怕是文件有中文的问题，去把文件名改了，这是我做过最错误的决定，因为正是因为改了文件名，后面执行一些命令按照源路径搜索不到了。以及整了半天才发现之前设置的环境配置没点保存……

因为看了一下视频以及评论区，好像没人的问题和我一样的，只能用deep seek来分析问题然后自己摸索着修改（ai太好用了你们知道吗

最后还是把文件名字改回来，然后删掉原来的文件，重新hexo init，接着npm install，就成功了（哭

后面基本上就很顺利了，今天上午上完课下午就回来想配置butterfly主题，但不知道为什么不行，总之先整理这些，我去hexo d上传了（笑
